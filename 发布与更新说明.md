# 专业就业地图网页版 PPT 发布与更新说明

## 发布目标

把当前目录 `web-presentation` 内的静态文件发布到一个固定线上地址。后续只要覆盖同一个线上目录或重新部署同一个项目，对方手里的链接不变，刷新后即可看到更新。

## 当前可发布目录

请发布本目录下的所有文件：

`outputs/web-presentation/`

必须包含：

- `index.html`
- `app.js`
- `presentation.css`
- `presentation/`

## 推荐发布方式

### 方式一：Netlify 拖拽发布

适合最快拿到公网链接。

1. 打开 Netlify 的 Sites 页面。
2. 新建站点或进入已有站点。
3. 把整个 `outputs/web-presentation/` 文件夹拖进去发布。
4. 第一次发布后会得到一个固定链接。
5. 后续修改 PPT 后，重新构建并拖拽同一个文件夹到同一个站点，原链接不变。

### 方式二：Vercel 项目发布

适合后续频繁更新。

1. 新建一个 Vercel 项目。
2. 上传或连接包含 `outputs/web-presentation/` 的仓库。
3. 将发布目录设置为 `outputs/web-presentation`。
4. 后续每次更新后重新部署同一个项目，链接不变。

### 方式三：公司服务器或对象存储

适合正式对外或内部分享。

1. 在服务器上固定一个目录，例如 `/ppt/employment-map/`。
2. 把 `outputs/web-presentation/` 中的所有文件上传到该目录。
3. 对外链接固定为类似 `https://公司域名/ppt/employment-map/`。
4. 后续更新时覆盖这个目录下的同名文件，链接不变。

## 后续更新流程

1. 修改 PPT 内容或样式。
2. 重新构建，生成新的 `outputs/web-presentation/`。
3. 覆盖发布到原来的线上位置。
4. 对方打开原链接并刷新即可看到新版本。

如果对方仍看到旧内容，请让对方强制刷新浏览器页面。
